常见问题解答

更多问题请查看Coding Plan FAQ
Lite 和 Pro 套餐的模型性能一样吗？
Lite 基础套餐与 Pro 高级套餐的模型性能一致，两者使用相同的模型资源与推理服务。
是否可以模型列表以外的模型？
Coding Plan 当前仅支持使用模型列表内的模型，使用其他模型将会报错。
Coding Plan 可以退款吗？
Coding Plan 服务一经购买不支持退订与退款。
额度消耗干净了怎么办？
Lite 套餐：5 小时额度或每周额度用完后，可等待自动恢复；每月额度用完后，可升级至 Pro 套餐，或等待下一订阅月续费后恢复。
Pro 套餐：5 小时额度或每周额度用完后，可等待自动恢复；每月额度用完后，需等待下一订阅月续费后恢复。
